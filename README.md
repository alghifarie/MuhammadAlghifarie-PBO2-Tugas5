# Aplikasi Perhitungan Kata

Aplikasi **Perhitungan Kata** adalah program berbasis Java dengan GUI yang memungkinkan pengguna menghitung jumlah kata, kalimat, dan paragraf dalam teks, mencari kata tertentu, serta menyimpan hasil ke file.

## Fitur

- **Hitung Otomatis**:
  - Jumlah kata, kalimat, dan paragraf dihitung secara otomatis saat teks diubah.
- **Pencarian Kata**:
  - Temukan berapa kali kata tertentu muncul dalam teks.
- **Simpan ke File**:
  - Simpan hasil perhitungan ke file teks.
- **Reset Input**:
  - Bersihkan teks untuk memulai dari awal.
- **Keluar dari Aplikasi**:
  - Tombol untuk keluar dari aplikasi dengan mudah.

## Cara Menggunakan

1. Jalankan aplikasi.
2. Ketik atau tempel teks ke dalam area input.
3. Lihat hasil perhitungan otomatis:
   - **Jumlah Kata**: Total jumlah kata dalam teks.
   - **Jumlah Kalimat**: Total jumlah kalimat berdasarkan tanda baca (.!?).
   - **Jumlah Paragraf**: Total jumlah paragraf berdasarkan pemisah baris.
4. Gunakan fitur pencarian:
   - Ketik kata yang ingin dicari di kolom **"Cari Kata"**.
   - Klik tombol **"Cari"** untuk melihat jumlah kemunculan kata.
5. Simpan hasil perhitungan ke file:
   - Klik tombol **"Simpan File"** dan pilih lokasi penyimpanan.
6. Klik tombol **"Ulang"** untuk menghapus teks dan memulai ulang.
7. Klik tombol **"Keluar"** untuk menutup aplikasi.

## Persyaratan Sistem

- **Java Development Kit (JDK)** versi 8 atau lebih baru.
- **Swing Library** (bagian dari Java Standard Edition).

## Instalasi dan Menjalankan Aplikasi

1. Pastikan JDK sudah diinstal di komputer Anda.
2. Clone repositori ini atau salin file program ke direktori lokal Anda:
   ```bash
   git clone https://github.com/username/PerhitunganKata.git
   ```
3. Compile program menggunakan perintah:
   ```bash
   javac PerhitunganKata.java
   ```
4. Jalankan aplikasi dengan perintah:
   ```bash
   java PerhitunganKata
   ```
